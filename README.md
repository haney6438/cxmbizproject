CXM 비즈 프로젝트

## 🧑‍💻 프로젝트 소개

> CXM 콘서트 기념 이벤트를 위해 제작한 비즈 키링 도안 제작 사이트입니다.<br>
> 콘서트 현장에서 비즈 키링 만들기 키트(비즈, 십자수 도구, 키링 부자재)와 함께 배포되었으며, <br>사용자가 원하는 도안을 선택하고 저장하여 비즈 키링 제작에 활용할 수 있도록 개발했습니다.

- 프로젝트명: CXM 비즈 프로젝트
- 개발 기간: 2026.01.01 ~ 2026.01.24
- 개발 인원: 1명 (개인 프로젝트)
- 주요 사용자: CXM 팬

<br>

## ✨ 주요 기능

### 1. 비즈 도안 조회
- 다양한 비즈 도안 목록 제공
- 도안 상세 페이지 확인 가능

### 2. 도안 이미지 저장
- 원하는 도안을 이미지 파일로 다운로드
- 다운로드 완료 시 Toast 메시지 제공

### 3. 커스텀 도안 제작
- 사용자가 직접 비즈 도안을 제작 가능
- 제작한 도안 저장 지원
- 이미지 파일을 도안으로 변환 기능 지원

### 4. 제작 가이드 제공
- 모달을 통한 비즈 키링 제작 방법 안내
